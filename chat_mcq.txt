Q1. What is the main purpose of a Binary Search Tree (BST)?
a) To store unsorted data
b) To perform depth-first search
c) To efficiently search for elements
d) To sort data
Answer: c) To efficiently search for elements

Q2. What is the time complexity of searching in a Binary Search Tree (BST)?
a) O(1)
b) O(log n)
c) O(n)
d) O(n log n)
Answer: b) O(log n)

Q3. Which graph traversal technique visits nodes in ascending order?
a) Depth-First Search (DFS)
b) Breadth-First Search (BFS)
c) In-Order Traversal
d) Pre-Order Traversal
Answer: c) In-Order Traversal

Q4. What is the primary use of a Graph?
